# Blockchain-Based Supply Chain Management System

A blockchain-enabled supply chain management (SCM) system designed to improve transparency, traceability, and data integrity across the supply chain lifecycle.

This project demonstrates how blockchain concepts can be applied to track products, transactions, and stakeholders in a decentralized and tamper-resistant manner.
